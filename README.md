# Actividad-formativa-3.-Implementaci-n-y-evaluaci-n-de-filtros-digitales

## Descripción
En esta actividad se implementaron filtros digitales en Python para analizar señales con ruido. Se diseñaron filtros pasa bajas, pasa altas y pasa bandas.

## Objetivo
Evaluar cómo los filtros afectan una señal en el dominio del tiempo y mejorar su calidad eliminando o resaltando frecuencias.

## Desarrollo
Se generó una señal con varias frecuencias y ruido. Después, se aplicaron los filtros y se compararon los resultados antes y después del filtrado.

## Resultados
- El filtro pasa bajas reduce el ruido.
- El filtro pasa altas resalta cambios rápidos.
- El filtro pasa bandas permite seleccionar un rango de frecuencias.

## Conclusión
Los filtros digitales son herramientas importantes para el análisis y mejora de señales.
